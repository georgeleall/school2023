Lesson 6: Working with Fonts, Text Blocks, Lists, and Tables

Summary: In this lesson we talked about different ways to style a web page. There is many different stylistic options when it comes to designing the website and different codes for different symbols. We also discussed how to arrange your web page in different ways.

One of the main things that I really enjoyed learning was the arrangement of multiple columns. I wonder if it will allow me to insert photos into each column and I can set up left and right banners for a web page and have content in between them. I learned a lot of stylistic tips in this lesson.

  • How to use boldface, italic, and special text formatting
  • How to tweak the font
  • How to use special characters
  • How to align text on a page
  • How to use the three types of HTML lists
  • How to nest lists within lists
  • How to create simple tables
  • How to control the size of tables
  • How to align content and span rows and columns within tables
  • How to use CSS columns

Lesson 7: Using External and Internal Links

Summary: In this lesson, we learned how to link internal pages, external pages, and emails in a web page. They showed an example of linking different locations on a web page using an anchor. This allowed for the user to click on the link and it will redirect them to a different position of the webpage.

We put this lesson to use in many different ways such as linking a web page to go from the top to the bottom and the bottom to the top of a web page. I find this to be useful because if you have a lot of content on your web page, it will seem like a drag for the user to scroll through it.

What You'll Learn:

  • How to use anchor links
  • How to link between pages on your own site
  • How to link to external content
  • How to link to non-Web documents such as PDF and Word documents
  • How to link to an email address
  • How to use window targeting with your links
  • How to style your links with CSS
  • How to add descriptions to links
  • Best practices for web page links
