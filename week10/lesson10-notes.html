Lesson 22: Using JavaScript Variables, Strings, and Array

Summary: In this lesson we discuss the different ways we can create different variables, how to name them, and what their uses are. We also discussed global and local variables. Global variables can be used throughout the entire script and local variables can pretty much only be used where they are created.

We also discussed the different operators that are used and their uses. We learned about using object methods and what they were. In the example, we used math and date. Where the math object was able to grab a random number and the date object is used to grab the date.

What You'll Learn:

  • How to name, declare, and assign variables
  • How to choose whether to use local or global variables
  • How to convert between different data types
  • How to create and use String objects
  • How to create and use arrays of numbers and strings
  • How to define, call, and return values from functions
  • How to use object properties and values
  • How to define and use object methods
  • How to use the Math object's methods
  • How to use new and this keywords to work with objects
  • How to use the Date object to work with dates

Bonus Items:

  • Choosing Variable Names and the rules used
    • Variable names can include letters of the alphabet, both upper- and lowercase. They can also include the digits 0-9 and the underscore (_) character.
  • Define global and local variables
    • Global variables have the entire script (and other scripts in the same HTML document) as their scope. They can be used anywhere, even within functions.
    • Local variables have a single function as their scope. They can be used only within the function they are created in.
  • List ways to assign value to variables
    • var bottles = 3;
    • bottles = 3;
  • List ways to increment and decrement variables
    • bottles = bottles + 1;
    • bottles = bottles - 1;
    • bottles += 1;
    • bottles -= 1;
  • Create a table of the JavaScript Operators, their description, and an example of each one
    • Operator Description
      + Concatenate (combine) strings
      + Add
      - Subtract
      * Multiply
      / Divide
      % Modulo (remainder)
      ++ Increment
      -- Decrement
  • List the JavaScript Data Types
    • Number
    • Boolean
    • String
    • Null
  • Define a String Object
    • String objects are what the string is stored as.
  • Define an Array
    • An array is a numbered group of data items that could be treated as a single unit.
  • Define a Function
    • A function is a group of JavaScript statements that can be treated as a single unit.
  • Describe how to return a value from a function
    • You can create a var that will equate to an equation and create a return statement.
  • Define an Object
    • Objects are like variables but can hold multiple values at once.
  • Define the Math Object
    • The Math object is a built-in JavaScript object that includes math constants and functions.
  • List methods to round and truncate
    • Math.ceil
    • Math.floor
    • Math.round
  • Provide a code example to generate a random number
    • function rand(num) {
      return Math.floor(Math.random() * num) + 1;
      }
  • Define the Date object
    • The Date object is a built-in JavaScript object that enables you to work more easily with dates and times. You can create a Date object anytime you need to store a date and use the Date object's methods to work with the date.
