Lesson 27: Working with Web-Based Forms

Summary:

In this lesson, we discuss how to incorporate user input into our code through HTML (specifically through forms). This could be useful when you would like to create a pop-up survey of a website you are continuously updating and need recommendations for.

The lesson shows us how to create these forms and goes over the different types of forms that HTML offers. It also shows us how to utilize the data from the forms and the forms themselves. They also showed us how to style these forms and the different ways they can look.

What You'll Learn:

  • How HTML forms work
  • How to create the front end of an HTML form
  • How to name pieces of form data
  • How to include hidden data in forms
  • How to choose the correct form input controls for different situations
  • How to validate form data
  • How to submit form data
  • How to use the form object with JavaScript
