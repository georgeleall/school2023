Lesson 12: Creating Layouts Using Modern CSS Techniques

Summary: In this lesson, we discussed the difference between fixed and liquid layouts. The difference between the is that a fixed layout is a fixed width in which the web page sits. A liquid layout adjusts its size depending on the size of the display.

I believe I would use a hybrid of both in my own website because I would want the website to adjust to the user's display but I would also want my website to look a certain way. Both have their uses, It depends on what the ambitions and goals are for the website appearance.

What You'll Learn:

  • How fixed layouts work
  • How liquid layouts work
  • How to create a fixed/liquid hybrid layout
  • How to create CSS flexible box layouts
  • How to use CSS grid layouts

Lesson 14: Using CSS Transformations and Transitions

Summary: In this lesson we discuss different ways to transform elements using CSS. You can scale an element with x and y or just scale one of those individually. You can scale an element using a scaling number.

Translate allows you to shift over an element. It will move it to wherever you want it to be. Although translate is very similar to a shift, it is very picky about verticality.

What You'll Learn:

  • How to transform elements by rotating, scaling, moving, and tilting
  • How to work with transformations in three dimensions
  • How to apply multiple transformations to one element
  • How to do simple animations with the transition property
