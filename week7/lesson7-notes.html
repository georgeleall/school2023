Responsive Web Design

Lesson 16: Understanding the Importance of Responsive Web Design

Summary: In this lesson, we discussed the important of Responsive Web Design (RWD). It is very important because you can code it once using CSS media queries and all the changes will be applied to whatever device you are using.

Another reason why it is important is because you can code it once and be done with the formatting. This will save plenty of time and help make it cleaner so you won't have to create scripts or code for it.

What You'll Learn:

  • What is responsive web design (RWD)
  • Why we need RWD
  • What is progressive enhancement
  • How to write HTML that's RWD ready

Lesson 17: Designing for Mobile Devices

Summary: Many people struggle to change when designing for mobile devices since they are building the webpage from their computer and not a mobile device. When testing these websites, it's encouraged to try the webpage on different sites and different devices.

Creating a web page that scales with a user's device will create an enjoyable experience for not just the user but for the designer since it will minimize the amounts of complaints. If the web design sucks it will reduce the traffic for the web page.

What You'll Learn:

  • How mobile design differs from standard web design
  • What the Mobile First design philosophy is
  • How Mobile First is important to RWD
  • How to build responsive tables and images for mobile devices
  • How to use CSS columns to create responsive designs without media queries

Lesson 18: Using Media Queries and Breakpoints

Summary: Media queries allow for the scaling of devices easier, it will be able to detect what kind of device is in use and determine its resolution. This helps the designer because it will adjust the webpage as intended for the resolution that is set in the query type that it falls in.

If a media query is within a certain range of pixels, it will adjust to whichever it falls into. It will scale the document to the settings that the designer set for it, whether it's a larger device or a smaller device, there is a preset for it and the content in the page will adjust itself.

What You'll Learn:

  • How to write a CSS media query
  • How to use different media types
  • How to create media query expressions
  • Understanding logical keywords in your queries
  • What CSS breakpoints are
  • How to define breakpoints
  • How to find the best breakpoints for your website
